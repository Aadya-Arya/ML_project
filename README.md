## Predicting student math performance using demographic and academic attributes.

An end-to-end Machine Learning project built using Python, Scikit-learn, and Flask to predict student math performance based on demographic and academic attributes.

### This project demonstrates a complete ML workflow including:
Data ingestion
Data transformation
Model training & evaluation
Hyperparameter tuning
Prediction pipeline
Flask web application
Logging & exception handling

### Tech Stack
Python
Pandas
NumPy
Scikit-learn
CatBoost
XGBoost
Flask
Matplotlib
Seaborn

### Features
Modular ML pipeline architecture
Data preprocessing using Scikit-learn pipelines
Model comparison with multiple regression algorithms
Hyperparameter tuning using GridSearchCV
Exception handling & logging system
Model and preprocessor serialization
Flask-based prediction interface

### Models Used
Linear Regression
Decision Tree Regressor
Random Forest Regressor
Gradient Boosting Regressor
AdaBoost Regressor
XGBoost Regressor
CatBoost Regressor
KNeighbors Regressor
